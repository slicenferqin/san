Context Steady State · M10

生命周期加固的重点是 replay/resume 不重复注入旧 ContextPacket

M10 把已有 digest/continuation/abort 保护和新增 ContextPacket replay contract 固定成测试。 目标是 session 恢复和后续轮次仍可 debug，但不会把旧 hidden packet 重新送进模型上下文。
